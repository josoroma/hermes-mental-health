# Scoring Guide — DSM-5-TR Measures

## PHQ-9 (Depression)
- Items: 9, scale 0-3 each
- Total range: 0-27
- Reverse scoring: None
- Severity thresholds: 0-4 None, 5-9 Mild, 10-14 Moderate, 15-19 Moderately severe, 20-27 Severe
- Data quality: Missing item 9 (self-harm) → flag; missing >=5 items → unscorable

## GAD-7 (Anxiety)
- Items: 7, scale 0-3 each
- Total range: 0-21
- Reverse scoring: None
- Severity thresholds: 0-4 Minimal, 5-9 Mild, 10-14 Moderate, 15-21 Severe

## PCL-5 (PTSD)
- Items: 20, scale 0-4 each
- Total range: 0-80
- Reverse scoring: None
- Provisional diagnosis: score >= 33
- Clusters: B (1-5) Intrusion, C (6-7) Avoidance, D (8-14) Cognition/Mood, E (15-20) Arousal

## WHO-5 (Well-Being)
- Items: 5, scale 0-5 each
- Raw range: 0-25
- Percentage: raw * 4 -> 0-100
- Clinical cutoff: <= 50 suggests depression screening

## PROMIS (T-score)
- Items: varies by bank
- Raw sum -> T-score lookup table
- T-score distribution: M=50, SD=10
- Severity: <55 None, 55-60 Mild, 60-70 Moderate, >70 Severe
